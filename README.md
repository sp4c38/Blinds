# Automated Blinds

This repository contains the project files for my automated window blinds, including the 3D-printable parts, circuit files, ESPHome configuration, and additional project material.

The files are provided for anyone interested in the project or who wants to learn from it. This is not a complete step-by-step build guide.

[Project showcase](https://www.space8.me/blinds.html)
